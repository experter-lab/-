# 实验六：555 多谐振荡器 Multisim 仿真清单

## 目标
用 NE555/LM555 搭建多谐振荡器，观察输出方波，并验证 R、C 改变对频率和脉宽的影响。

## 元件
- NE555 或 LM555 定时器：1 个
- 直流电源 VCC：5 V
- GND：1 个
- R1：1 kΩ
- R2：10 kΩ 电位器，或先用固定电阻 10 kΩ
- C1：10 uF，之后改为 0.1 uF 做对比
- C2：0.01 uF，接控制端滤波
- LED + 限流电阻 330 Ω，可选
- Oscilloscope 示波器：观察 3 脚输出和电容电压

## 555 引脚接线
- 1 GND -> 地
- 8 VCC -> +5 V
- 4 RESET -> +5 V
- 5 CTRL/CV -> 0.01 uF -> 地
- 2 TRIG 与 6 THRES 短接
- 7 DISCH -> R1/R2 连接点
- 3 OUT -> 示波器 CH A；也可串 330 Ω 接 LED 到地

## 多谐振荡器连接
- R1：+5 V -> 7 脚
- R2：7 脚 -> 2/6 脚
- C1：2/6 脚 -> 地
- 3 脚输出 -> 示波器 CH A
- 2/6 脚电容电压 -> 示波器 CH B

## 理论估算
周期：T ≈ 0.693 * (R1 + 2R2) * C
频率：f ≈ 1 / T
高电平时间：tH ≈ 0.693 * (R1 + R2) * C
低电平时间：tL ≈ 0.693 * R2 * C

当 R1=1kΩ, R2=10kΩ, C=10uF：
T ≈ 0.693*(1k+20k)*10uF ≈ 0.1455s
f ≈ 6.87Hz

当 C=0.1uF：
T ≈ 1.455ms
f ≈ 687Hz

## 仿真设置建议
- Analysis: Transient Analysis
- 起始时间：0 s
- 结束时间：C=10uF 时用 1 s；C=0.1uF 时用 10 ms
- 最大步长：周期的 1/200 左右，C=10uF 可用 0.5 ms，C=0.1uF 可用 5 us

## 应观察现象
- 3 脚输出为方波，电压接近 0 V 与 5 V 切换。
- 2/6 脚电容电压在约 1/3VCC 和 2/3VCC 之间充放电。
- 增大 R2 或 C1，频率下降；减小 R2 或 C1，频率升高。
