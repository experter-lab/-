- ✅ 实现三级安全检查函数 `_check_three_level_safety()`

- ✅ 在状态发布中添加授权信息
- ✅ 重新编译并验证

**三级安全机制**:
```
第一级：软件急停 (emergency_stop)
  - 默认启用
  - 可通过服务切换

第二级：硬件急停 (hardware_estop_detected)
  - 检测物理急停按钮
  - 通过MAVLink心跳监测

第三级：控制授权 (control_authorized)
  - 必须显式授权才能控制
  - 默认60秒后自动过期
  - 剩余10秒时发出警告
```

**使用方法**:
```bash
# 授权控制（60秒）
ros2 service call /chassis_bridge/authorize_control std_srvs/srv/SetBool "{data: true}"

# 取消授权
ros2 service call /chassis_bridge/authorize_control std_srvs/srv/SetBool "{data: false}"
```

**收益**:
- 🛡️ 安全性提升：防止误操作造成危险
- ⏱️ 自动保护：授权自动过期，无需手动关闭
- 📊 状态可见：实时显示授权状态和剩余时间
- 🔒 多层防护：三级检查，层层把关
