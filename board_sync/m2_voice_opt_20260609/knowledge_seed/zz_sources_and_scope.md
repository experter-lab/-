# Knowledge source registry and scope

This lightweight patient-facing knowledge base is for general medication safety reminders, not diagnosis or individualized prescriptions.

Primary reference sources checked for scope and safety framing:
- MedlinePlus Medicines: https://medlineplus.gov/medicines.html
- FDA Medicine and Pregnancy: https://www.fda.gov/consumers/womens-health-topics/medicine-and-pregnancy
- National Institute on Aging, Taking Medicines Safely as You Age: https://www.nia.nih.gov/health/medicines-and-medication-management/taking-medicines-safely-you-age

Operational rule:
- Use current delivery task data first for the patient's own medicines.
- The knowledge base only adds general safety reminders.
- If dose, timing, meal relation, pregnancy/child/elderly risk, allergy, or multi-drug interaction is unclear, escalate to nurse, doctor, or pharmacist.
- Do not use random GitHub medical content as factual source for patient advice. GitHub can be used for RAG formatting patterns only.
